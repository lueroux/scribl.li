# Font Setup Instructions

The landing page uses the same Inter variable font as the main Scriblli app.

## Font Files Copied

The following fonts have been copied from the main app:
- `inter-variablefont_opsz,wght.ttf` - Inter variable font (regular)
- `inter-italic-variablefont_opsz,wght.ttf` - Inter variable font (italic)

## Font Location

Fonts are located in: `/fonts/`

## Usage

Fonts are automatically loaded via `@font-face` declarations in `css/scriblli-theme.css`

The design system uses:
- **Primary Font**: Inter (variable weight 100-900)
- **Brand Color**: Scriblli Green `hsl(83, 47%, 64%)`
- **Design Style**: Bauhaus-inspired, geometric, functional
- **Border Radius**: Minimal (0.125rem) to Geometric (0.25rem)
- **Shadows**: Minimal and flat only

## Design System Match

This landing page now matches the main Scriblli app design system including:
- Color palette (primary green, neutral grays)
- Typography (Inter variable font)
- Component styling (buttons, cards, inputs)
- Spacing and layout patterns
- Transitions and animations
